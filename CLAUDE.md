## Design Context

This is Sanath's personal site — a calm, ink-on-paper developer portfolio (see `PRODUCT.md` and `DESIGN.md`). One signature moment: a looping ASCII-art ocean wave behind the hero title; everything below it is flat, static, and content-first (about + projects now, blog/photos later). Near-monochrome palette (Ink & Paper), sans-led type with a mono accent and sparing serif. Named anti-reference: the rest of eloyb.design (great hero, too "fancy" after). Read `PRODUCT.md` / `DESIGN.md` before making UI changes.
